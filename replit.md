# Class XII Computer Science Practical File

## Project Overview
This is a comprehensive Class XII Computer Science (083) practical file for Session 2025-26, containing all required programs as per CBSE syllabus.

## Project Structure

### Part A: Python Programs (20 Programs)
Located in `Program/Python Programs/`
- **File Handling (8 programs)**: Text files, binary files, CSV files
- **Functions & Exception Handling (4 programs)**: Recursion, try-except
- **Data Structures - Stack (2 programs)**: Stack implementation and applications
- **Random & Miscellaneous (2 programs)**: Random module applications
- **CSV/Binary Applications (2 programs)**: Student records management
- **Combined Tasks (2 programs)**: Integrated file operations

### Part B: SQL Queries (10 Queries)
Located in `Program/SQL Queries/`
- SELECT, UPDATE, DELETE, ALTER operations
- Aggregate functions (COUNT, MAX, MIN, AVG)
- GROUP BY and LIKE clauses
- Student table database operations

### Part C: Python-MySQL Connectivity (5 Programs)
Located in `Program/Python-MySQL/`
- CRUD operations using mysql.connector module
- Insert, Display, Search, Update, Delete records
